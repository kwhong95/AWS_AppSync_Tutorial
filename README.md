# AWS_AppSync Series

## 0. AWS AppSync Settings with Amplify

### 1. Install AWS-Amplify

```shell
npm i -g @aws-amplify/cli
```

```shell
amplify configure
```

### 2. Set Up AWS-Amplify for a Project

```shell
amplify init
```

```shell
amplify add api
```

### 3. Creating a client

```shell
yarn add aws-appsync aws-appsync-react react-apollo 
```
